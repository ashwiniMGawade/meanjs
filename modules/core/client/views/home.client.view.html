Hourly Snapshots Every 3 hours during business hours (10AM to 4PM) 3 days
Daily Snapshots Every night 12AM - 7 days a week 2 weeks
Weekly Snapshots Once in a week. (Sunday Midnight) Daily snapshot will not be taken on this day. 12 weeks

Answer - Permissions are assigned to Project Shares based on the group membership. Normally there will be three security groups namely CC, PL and DV. Only CC (apart from administrators) will have full control on the share. PL group will be having Modify rights and DV group will be having only Read permission which can be changed wherever required by the CC.

Go to Start, Computer and click on Network on left side pane. In the tool bar click on Search Active Directory:

Select Active Directory from the drop down menu and type the group name which you want to manage in the Name Field and click on Find Now. Now the group will be shown in the bottom pane of the search window. (Sequence numbered)

Double click on the group name to open the properties window. Clicking on the ADD button, which will open up a menu where you can add the user to this group as shown below. (Sequence Numbered)

Now this user is added to the group and the permissions will get effective when the user re login to the domain.

Points -

  • Never remove access for Administrators group from any share. If administrators group ID is removed access, recovery of deleted files will not be possible by the Admins.
  • Do not give permissions to open DLs which have the all the users of the domain or list of all employees for a given location.
  • Do not ever give permissions to group such as “Everyone” or “Domain Users”.
  • Make sure that you are providing access to the right users; always give permission to the user based on their mail id, not with their display name.
  • Give users only the necessary privilege to carry out their job, If a user in DV group need write access on to only one folder, don’t add him to PL group, rather give him write/modify access only on that folder.
  • At any point, if you have any doubt, please call CCD to clarify yourself.
